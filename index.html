TextMesh Pro 显示中文字符的方法

2020-12-05

这两天想写一个用于Unity项目中的对话系统时接触到了TextMesh Pro这个玩意儿，它相比于普通的Text组件强了不少，但是居然不默认支持中文。咋办呢？

上网查了查，发现解决的方法很简单。这里我来做个演示：

使用Font Asset Creator

首先打开Font Asset Creator这个工具。

Source Font File 就是选用的字体文件。

Sampling Point Size 可以自己选择数值，不过一般情况下选Auto Sizing 就可以。

Padding 的数值会影响描边或者阴影的效果，一般保持默认。

Packing Method 选快速就可以。

Atlas Resolution 这个尺寸根据字符的数量等因素决定。中文字符比较多所以选择大一些。

Character Set 是选择用于生成的字符集，根据需求来选择。这里我有一个现成的文件（包含数千个中文字符的文本文件），所以选择Characters from File这个选项。假设项目中所需字符数量少，那么选择Custom Characters也行。

Render Mode 渲染方式。

生成并保存Font Asset

这些选项选好了之后，点击Generate Font Atlas开始生成Font Asset文件。因为字数比较多，所以生成速度可能会有些慢，耐心等待。

渲染结束后就可以在右边看到成果。

值得注意的是，左下角的方框里包含了一些信息，其中Point Size这个数值会影响显示时的清晰度，还有Missing characters里面可以看到哪些字符生成失败（原因大概是字体里面不包含这些字符）。

确认无误后，点击Save或者Save as进行保存。

应用Font Asset至TMP

保存之后，到TMP的Inspector里面修改Font Asset为刚刚保存的那个。

成功显示中文。
